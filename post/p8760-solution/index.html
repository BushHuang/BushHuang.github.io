P8760 填空问题 题解

· 2023-01-09 · # OI # 题解

这是一个寻找规律题。

首先，我们制作一个可以寻找前 nnn 个斐波那契数列的数个位的程序：

#include<bits/stdc++.h>
using namespace std;
int a=1,b=1,c;         //由于只取个位，所以不需要考虑爆long long的问题
long long n;
int main()
{
	cin>>n;  
	cout<<"1 1 ";           //前两个数
	for(int i=1;i<=n;i++)
	{
		c=(a+b)%10;     //每次只取个位
		cout<<c<<" ";
		a=b;
		b=c;
	}
	return 0;
}

很明显，这是一个周期为 606060 的数列，其中“777”出现了 888 次。

计算可得答案：
